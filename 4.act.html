3. Difference between Paragraph Tag and Article Tag

Paragraph: The paragraph tag is used for adding individual blocks of text to a webpage. It focuses on text without any structural meaning.

Does not add semantic meaning to the text other than marking it as a paragraph.

Typically used for general text and can appear multiple times in a document.

Example:

This is a paragraph 1 explaining HTML basics.

This is another paragraph 2 about CSS styling.


Article: The article tag represents a standalone section of content that is meaningful on its own, like a blog post or a news piece.

Adds semantic meaning, indicating the content is a distinct, independent piece of content.

Contains structured content like headings, paragraphs, and other elements related to the topic.

Example:

Topic of the article

This is the content of the article.

4. Difference between Anchor Tag and Navigation Tag

Anchor: The anchor tag creates hyperlinks, enabling navigation between web pages or within the same page.

It is an inline element.

Example:

Click here to visit Google

Navigation Tag: By using multiple anchor tags, you can create a navigation menu for a website.

It is a semantic block-level element used to group primary navigation links for a webpage.

Example:

Home | About | Contact

7. Difference between Strong Tag, Bold Tag, and Emphasis Tag

Strong: The strong tag is used to highlight important content, adding emphasis that indicates significance.

Example: This is important information.


Bold: The bold tag adds bold styling to text without necessarily implying importance.

Example: This text is bold.


Emphasis: The emphasis tag indicates stress or importance, typically displayed in italics.

When any text enclosed with em tag is displayed on a webpage, the word will appear in italics, indicating that it is being emphasized.
